# Ecommerce Monolito

Aplicação que simula um back-end de um ecommerce. Desenvolvido para estudo do framework Spring Boot.

## Tecnologias

No projeto é usado as seguintes tecnologias:

- Java (JDK 11)
- Spring boot (Versão 2.1.7.RELEASE)
- Maven
- Docker
- Swagger
- Mysql (Versão 5.7)
